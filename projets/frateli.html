Frateli

Algorithmes de recommandation pour le parrainage d'étudiants.

Code source : Github

Le projet

Optimiser le matching parrain-filleul pour Frateli afin de prévoir si un matching va réussir ou échouer grâce à un outil algorithmique d'aide à la décision pour améliorer le taux de réussite du parrainage.

Le partenaire

Frateli développe et encadre le parrainage de jeunes étudiants boursiers à haut potentiel par des professionnels.
